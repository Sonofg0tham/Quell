# Quell Brand Identity

Direction: still water, sealed - calm, discreet, watertight. Quell
dampens the fire; where Ward is cut iron and ember, Quell is cool
water and silence.

Fonts:
- Display: Cabinet Grotesk (via Fontshare)
- Body: Satoshi (via Fontshare)
- Mono: Fira Code (via Google Fonts)

Palette (HSL):
- Background: hsl(200 14% 8%)      /* cool slate near-black, never #000 */
- Foreground: hsl(200 12% 92%)     /* off-white, never pure #FFF */
- Accent: hsl(192 70% 46%)         /* still-water aqua - #23A7C7 */
- Border: hsl(200 12% 20%)
- Muted: hsl(200 8% 60%)

Light-mode counterparts:
- Background: hsl(195 25% 97%)
- Foreground: hsl(200 14% 11%)
- Border: hsl(198 15% 86%)

Radius scale: 0.5rem (soft and contained - calm, not cut)

Signature detail: the redaction bar - a solid rounded horizontal bar
standing in for anything secret, drawn in the accent aqua. It fills
the Q's counter in the mark (the Sealed-Q), replaces the E's middle
arm in the wordmark, and recurs anywhere a secret would appear
(echoing Quell's own {{SECRET_xxx}} placeholders). Secondary motif:
double braces {{ }} as frames.

VS Code gallery banner: dark, #10171C.

product Craig has built.
